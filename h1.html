Esta es la primera parte de mi experiencia en HTML.

HTML es el lenguaje estandar para crear paginas web, describe la estructura de esta y consta de una serie de contenidos.

Ademas los elementos etiquetan piezas de elementos.

¿Que es un elemento HTML?

El elemento HTML es todo, desde la etiqueta del principio hasta la del final.

Como por ejemplo "h1 mi primer encabezado h1".

Navegadores web.

El propósito de un navegador web es leer documentos HTML y mostrarlos correctamente.

Un navegador no muestra las etiquetas HTML, pero las usa para determinar cómo mostrar el documento

Documentos HTML.

Todos los documentos HTML deben comenzar con una declaración de tipo de documento: "!DOCTYPE html".

El documento HTML en sí comienza "html" y termina con "/html".

La parte visible del documento HTML está entre "body" y "/body".

Encabezados HTML.

Se definen con las etiquetas desde "h1" hasta "h6".

Parrafos HTML.

Los parrafos se definen con la etiqueta "p".

Enlaces HTML.

Los enlaces HTML se definen con la etiqueta "a".

El destino del enlace se especifica en el href atributo

Los atributos se utilizan para proporcionar información adicional sobre los elementos HTML.

Imagenes HTML.

Las imagenes en HTML se definen con la etiqueta "img".

El archivo de origen "src" y el texto alternativo "alt", width y height se proporcionan como atributos.

Ver código fuente HTML.

Haga clic derecho en una página HTML y seleccione "Ver código fuente"

Esto abrirá una ventana que contiene el código fuente HTML de la página.
