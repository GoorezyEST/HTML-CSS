HTML Basico

Los documentos HTML

Todos los documentos HTML deben comenzar con una declaración de tipo de documento: !DOCTYPE html.
El documento HTML en sí comienza html y termina con /html.
La parte visible del documento HTML está entre body y /body.
La declaracion !DOCTYPE permite definir el tipo de documento y debe usarse solo una vez.

Los encabezados

Se definen con etiquetas desde h1 hasta h6.
Desde el menos importante hasta el mas.

Los parrafos

Los parrafos se definen con la etiqueta p.

Los enlaces

Los enlaces se definen con la etiqueta a.

Las imagenes

Las imagenes se definen con la etiqueta img.
El archivo de origen src y el texto alternativo alt.

Elementos HTML

Los elementos

Los elementos son todas las etiquetas desde el principio hasta el final.
Por ejemplo h1 Mi titulo /h1.
Algunos elementos no tienen contenido como por ejemplo la etiqueta br.

Elementos HTML anidados.

Los elementos se pueden anidar, significa que puede haber elementos dentro de otros elementos.
Por ejemplo dentro de la etiqueta html esta el body, y dentro del body el h1.

Atributos HTML

Los atributos

Todos los elementos HTML pueden tener atributos
El href atributo especifica la URL de la página a la que va el enlace
El src atributo de imgespecifica la ruta a la imagen que se mostrará
Los atributos width y height de proporcionan información sobre el tamaño de las imágenes.
El alt atributo de img proporciona un texto alternativo para una imagen.
El style atributo se usa para agregar estilos a un elemento, como color, fuente, tamaño y más
El lang atributo de la html etiqueta declara el idioma de la página web.
El title atributo define alguna información adicional sobre un elemento.

Encabezados HTML

Los encabezados

Los encabezados se definen con las etiquetas h1 hasta h6.

Son muy importantes

Los motores de búsqueda utilizan los encabezados para indexar la estructura y el contenido de sus páginas web.
Los usuarios a menudo hojean una página por sus títulos.
Es importante utilizar encabezados para mostrar la estructura del documento.

Parrafos HTML

Los parrafos y sus etiquetas.

Un parrafo esta definido por el elemento p.
Dentro de el se pueden usar distintas etiquetas.
Esta hr para marcar un cambio de tema.
Luego br para insertar un salto de linea
Y pre para definir el pre-formato de un texto.

Estilos HTML

Los estilos

El atributo style se usa para agregar estilos a un elemento.
Como el color, la fuente o el tamaño.
Su etiqueta es: etiqueta style="propiedad:valor".

Utilizar background-color para el color de fondo
Usar color para colores de texto
Usar font-family para fuentes de texto
Usar font-size para tamaños de texto
Uso text-align para alineación de texto
